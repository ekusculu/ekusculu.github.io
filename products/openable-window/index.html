Openable Marine Window

ISO 3903 compliant

Application

Bridge deck, accommodation, service areas.

Description

Karaca openable marine windows are designed for superstructure integration in compliance with ISO 3903, providing ventilation while maintaining weather tightness and structural reliability.

Key Features

  • Heavy-duty marine hinges
  • Adjustable locking mechanism
  • Tempered or laminated glass
  • Weather-tight gasket system

Glass & Performance Options

  • Double glazing (thermal insulation)
  • Electrically heated glass
  • Bonded glazing option
  • Solar control integration
  • Project-specific U-value configuration

Installation Options

  • Welded
  • Bolted
  • Bonded
  • Clamp type
