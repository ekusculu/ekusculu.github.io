Bonded Glass System + Pantograph Wiper

Structurally Bonded Marine Glazing

Karaca bonded glazing systems are fixed directly to the vessel structure using certified structural silicone systems, eliminating traditional framing and providing a flush exterior appearance.

Key Features

  • Frameless aesthetic design
  • Black ceramic frit (UV protection)
  • SGP interlayer option
  • High-performance laminated build
  • Thermal / heated configurations available

Designed in line with ISO 11336-2, ISO 3902 and ISO 3434 where applicable.

Optional Configurations

  • Ballistic-resistant glass option
  • EMI shielding film
  • Custom marine black border

Pantograph Wiper

  • Dual-arm system
  • Large wiping coverage
  • Adjustable wiping angle
  • Heavy-duty 24V DC motor
  • Integrated washer system
