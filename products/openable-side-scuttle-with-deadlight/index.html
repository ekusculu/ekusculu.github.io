Openable Side Scuttle with Deadlight

ISO 1751 compliant

Application

Accommodation areas, engine rooms, control spaces.

Description

Karaca openable side scuttles are manufactured in compliance with ISO 1751 for watertight marine applications. The system includes an internal hinged steel deadlight (storm cover) providing additional protection under heavy weather conditions.

Key Features

  • Marine-grade aluminium, steel, stainless steel or brass frame
  • Toughened or laminated safety glass
  • Fully watertight construction
  • Internal hinged deadlight
  • Class-approved production

Optional Configurations

  • Double glazed thermal configuration
  • Electrically heated glass (ISO 3434 compliant zones)
  • Non-magnetic execution (military applications)
  • AISI 316L stainless steel
  • Custom RAL coating

Installation Options

  • Welded
  • Bolted
  • Clamp type
